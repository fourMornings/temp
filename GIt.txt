GIT


- Переходим в необходимую директорию

- создаем файл
	>touch readme.txt

- инициализируем проект
	>git init

- проверяем статус файлов
	>git status

- добавляем файлы под трекинг
	>git add *

- делаем комит
	>git commit -a -m"Init"
		-a  -- 
		-m  -- means message 

- проверяем изменения
	>git log

- просмотр разницы между файлами перед комитом
	>git diff

- добавляем указание на удаленный репозиторий
	>git remote add origin https://github.com/fourMornings/temp.git

- отправляем мастер на удаленный репозиторий
	>git push -u origin master
