GIT


- Переходим в необходимую директорию

- создаем файл
	>touch readme.txt

- инициализируем проект
	>git init

- проверяем статус файлов
	>git status

- добавляем файлы под трекинг
	>git add *

- делаем комит
	>git commit -a -m"Init"
		-a  -- 
		-m  -- means message 

- проверяем измененич
	>git log

- просмотр разницы между файлами
	>git diff
